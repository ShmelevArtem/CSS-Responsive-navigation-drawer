Responsive navigation drawer

Небольшое красивое меню уменьшающее размер при нажатии на кнопку.
Использовался html и css и javascript.

Оригинальное видео, чтобы создать такое:
https://www.youtube.com/watch?v=Az_TbiPAyDs&t=612s

Мною меню переведено на русский язык, немного улучшена CSS код, для анимации кнопок, JavaScript вынесен в отдельный файл.
